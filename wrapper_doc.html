My C Wrapper Documentation

What is the Wrapper?

My C Wrapper is a library that provides additional functionality to Bash scripts, such as handling key input, mouse input, displaying windows, drawing shapes on the window, and displaying text. It is built using the C programming language and can be compiled with a basic C compiler.

How to Include the Wrapper in Your Bash Scripts?

To include the wrapper in your Bash script, you first need to compile the wrapper program into a shared library (e.g., my_c_wrapper_program.so). Then, you can use the LD_LIBRARY_PATH environment variable to specify the path to the shared library when running your Bash script, like this:

LD_LIBRARY_PATH=/path/to/my_c_wrapper_program/ ./your_bash_script.sh

Wrapper Functionality

The My C Wrapper library provides the following functions:

  • init_window: Initializes a window for drawing.
  • close_window: Closes the window.
  • get_key_input: Waits for a key press event and returns the key code.
  • get_mouse_input: Waits for a mouse event and returns the mouse coordinates.
  • display_window: Displays the window on the screen.
  • draw_rectangle: Draws a rectangle on the window.
  • draw_circle: Draws a circle on the window.
  • draw_text: Draws text on the window.

How to Use the Wrapper Functions?

To use the wrapper functions in your Bash script, you first need to include the shared library using the source command, like this:

source /path/to/my_c_wrapper_program/my_c_wrapper_program.sh

Then, you can use the wrapper functions in your script, like this:

	# Initialize the window
	init_window 640 480

	# Draw a rectangle on the window
	draw_rectangle 10 10 100 100 "#ff0000"

	# Display the window on the screen
	display_window

	# Wait for a key press event and print the key code
	key=$(get_key_input)
	echo "Key pressed: $key"

How to Compile Your Bash Script with the Wrapper?

To compile your Bash script with the wrapper, you can use the shc command, like this:

shc -r -f your_bash_script.sh -o your_bash_script

This will compile your Bash script with the wrapper and create an executable file called your_bash_script.
